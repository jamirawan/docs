# Paragraphs

*Notes:*
- To create paragraphs, use a blank line to separate one or more lines of text.
- Don't ident paragraphs with spaces or tabs

***Markdown***

```md
This is the first paragraph.

This is the second paragraph
```

***Rendered output:***

This is the first paragraph.

This is the second paragraph
