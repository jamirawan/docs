# Theme

## Theme structure

Theme file structure will be like this
```
.
├── Gemfile
├── LICENSE.txt
├── README.md
├── _includes
├── _layouts
│   ├── default.html
│   ├── page.html
│   └── post.html
├── _sass
├── assets
└── irawan.gemspec
```
